# 股票對決遊戲（30 位 AI 對手）

這是一款「玩家 vs 30 位 AI 交易員」的股票對決遊戲。核心玩法包含：

- 玩家與 30 位 AI 在同一市場中交易，爭奪資產排名。
- AI 具備不同的風格與情緒，會互相競爭甚至聯盟。
- 市場包含事件系統（重大新聞/黑天鵝/季節效應）。

## 近期優先功能（MVP+）

1. **選擇權交易**：Call/Put 多空策略與槓桿模型。
2. **AI 個性化對話**：賦予 AI 交易風格與互動對話。
3. **黑天鵝事件**：隨機事件強化風險管理與刺激感。
4. **成就系統**：提升遊戲目標感與成就驅動。
